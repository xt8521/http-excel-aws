## Clone the project

```
$ git clone https://go.googlesource.com/example
$ cd http-excel-aws
```

## [Run application](./)

```
$ cd http-excel-aws
$ go run .
```

A trivial client http.

Topics covered:

* Extract data from excel ([flag](//golang.org/pkg/flag/))
* Signer AWS V4 ([log](//golang.org/pkg/log/))
